Installation:

npm install

To Run Test Suite:

npm test

To Start Server:

npm start

To Visit App:

localhost:3000/ 

Stack:\
1)JavaScript\
2)Html / Css\
3)React\
4)Mobx\
5)React-router-dom v6\
6)Tailwind css\
7)Firebase - for registration authorization - password must contain at least 6 characters\
8)Axios\
